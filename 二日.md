### 三、核心参数：自定义渲染行为
`MarkdownBody` 提供丰富的参数自定义样式、交互，以下是常用参数：

| 参数 | 作用 | 示例 |
|------|------|------|
| `data` | 必传，Markdown 文本字符串 | `data: "# 标题"` |
| `styleSheet` | 自定义文本样式（字体、颜色、大小等） | 见下方示例 |
| `onTapLink` | 拦截链接点击事件（替代默认跳转） | 见下方示例 |
| `imageBuilder` | 自定义图片渲染（替代默认 Image 组件） | 见下方示例 |
| `selectable` | 是否允许文本选中复制 | `selectable: true` |

#### 示例 2：自定义样式（styleSheet）
```dart
MarkdownBody(
  data: markdownData,
  // 自定义样式表，覆盖默认样式
  styleSheet: MarkdownStyleSheet(
    // 一级标题样式
    h1: const TextStyle(
      fontSize: 24,
      fontWeight: FontWeight.bold,
      color: Colors.blue,
      height: 1.5,
    ),
    // 加粗文本样式
    strong: const TextStyle(
      fontWeight: FontWeight.bold,
      color: Colors.red,
    ),
    // 链接文本样式
    link: const TextStyle(
      color: Colors.green,
      decoration: TextDecoration.underline,
    ),
    // 代码块样式
    codeblock: const TextStyle(
      fontSize: 14,
      backgroundColor: Color(0xFFF5F5F5),
      fontFamily: "monospace",
    ),
    // 边距配置（如列表、引用的缩进）
    listIndent: 20, // 列表缩进
    blockquote: const TextStyle(color: Colors.grey), // 引用文本颜色
  ),
)
